### Lessons Learned
- WhatsApp only allows ONE Web session per account — never run two gateways simultaneously
- `docker stop` is NOT permanent in Coolify — it auto-restarts. Use `docker rm -f` to permanently remove
- All hardcoded Mac paths (`/root/.openclaw`) must be converted to `/root/.openclaw` for Docker
- Gateway must bind to `0.0.0.0` (not `loopback`) when running in a container for external access
